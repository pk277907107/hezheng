## 如何运行

直接用浏览器打开 `index.html` 即可查看网站。

### 使用本地服务器运行（推荐）

如果您有 Python，可以运行：

```bash
# Python 3
python -m http-server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问 `http://localhost:8000`

如果您有 Node.js，可以使用：

```bash
npx serve .
```
